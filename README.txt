Polecenie uruchomienia: python3 play.py
LUB (jeśli domyślny interpreter to Python3): python play.py
Do uruchomienia wymagana jest instalacja Python 3.x.
Python 2.x nie obsługuje użytego w grze kodu.
Aby zagrać w trybie pełnoekranowym należy przy uruchomieniu podać argument: fullscreen ("python3 play.py fullscreen")

Pliki źródłowe:
- play.py - gra z obsługą menu, pełnego ekranu i kustomizacją rozgrywki
- sicore.py - jedynie uruchamia grę z domyślnymi parametrami
- siclasses.py - biblioteka klas użytych w grze
- resources - folder z obrazami używanymi w grze
- leaderboard.txt - przechowuje wyniki z każdej rozgrywki

Opracowano i przetestowano na Python 3.5.3 oraz pygame 1.9.4.
Rozpowszechnianie plików źródłowych i ich zawartości jedynie za zgodą autora.
